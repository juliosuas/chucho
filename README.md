# 🎮 Chucho y las Aventuras del Río

An educational math game for kids ages 5-8, inspired by classic 2000s educational games.

## How to Play

Help Chucho cross the river by answering math questions correctly! Each correct answer makes Chucho jump to the next stone. Wrong answers make him splash into the water!

### Levels
- **Nivel 1**: Easy addition (1-10)
- **Nivel 2**: Easy subtraction (1-10)  
- **Nivel 3**: Mixed addition & subtraction (1-20)
- **Nivel 4**: Basic multiplication (tables 1-5)
- **Nivel 5**: Everything mixed!

## Features
- 🎨 All graphics drawn with Canvas 2D API (no external images)
- 🎵 Sound effects via Web Audio API
- 📱 Responsive design (works on mobile)
- 🏆 High scores saved locally
- 🌿 Animated jungle environment with butterflies, dragonflies, frogs, and fish

## Tech Stack
- Pure HTML5 Canvas + vanilla JavaScript
- No libraries or frameworks
- 60fps game loop with requestAnimationFrame

## Running
Just open `index.html` in a browser. No build step needed.
